# Security Policy

## Scope

This repository contains the source for a static personal website and writing.
It is not intended to process, store, or transmit sensitive data.

No secrets, credentials, API keys, or private information should ever be committed
to this repository.

## Reporting a Vulnerability

If you believe a security issue exists in this repository (for example:
accidental disclosure of sensitive information, unsafe configuration, or
supply-chain concerns), please report it responsibly.

**Contact:**
mert.satilmaz@gmail.com

Please include:
- a clear description of the issue
- affected files or paths
- steps to reproduce (if applicable)

## Disclosure Expectations

This repository is maintained on a best-effort basis.

There are no formal SLAs, but responsible reports will be reviewed and addressed
as reasonably and promptly as possible.

## Out of Scope

The following are considered out of scope:
- hypothetical issues without a concrete impact
- vulnerabilities in third-party platforms or services
- general critique of content or opinions expressed in posts

## Thanks

Responsible disclosure helps keep public repositories safe and trustworthy.
Thank you for reporting issues thoughtfully.
